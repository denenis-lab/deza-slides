05 / 11

Основы

Окружение. CLI. Структура.

Environment

uv — рекомендованный менеджер. Минимум: .venv/ + streamlit_app.py

CLI

uv run streamlit run app.py для запуска. config.toml для настроек

Code Organization

Файл <1000 строк. UI отдельно, логика в utils/

Markdown

:material/icon: вместо эмодзи. :red[text] для цвета. LaTeX для формул

06 / 11

Интерфейс

UI и дизайн. 5 skills.

Layouts

Sidebar для фильтров. Макс. 4 колонки. st.dialog для фокуса

Widgets

2‑5 опций → segmented_control. Много → selectbox

Design

Material icons. Sentence case. Без лишних dividers

Themes

4 цвета в config.toml. Light + Dark. Без кастомного CSS

07 / 11

Производительность

Данные. Скорость. 4 skills.

Performance

@st.cache_data для данных. @st.fragment для изоляции. Условный рендеринг вместо tabs

Session State

setdefault() на старте. Key автосинхронизация. Не мутировать cache_resource

Data Display

Нативные чарты первыми. Altair для сложного. column_config по необходимости

Dashboards

border=True для карточек. horizontal=True для KPI. Sparklines в метриках

08 / 11

Продвинутое

Чат. Multipage. Компоненты.

Chat UI

История в session_state. write_stream() для streaming. Файлы и аудио

Multipage

app_pages/ папка. Общее состояние в main. 3‑7 страниц → top nav

Custom Components

Inline для простого, packaged для сложного. Shadow DOM + CSS‑переменные

Snowflake

st.connection(). TTL‑кэширование. Параметризованные запросы, не f‑string
